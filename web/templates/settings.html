环境变量配置

以下配置需要在 .env 文件中设置：

  • ANTHROPIC_API_KEY - Claude API 密钥 (用于智能摘要)
  • SERVERCHAN_SENDKEY - Server 酱推送密钥
  • GEWE_BOT_URL - Gewebot 服务地址
  • GEWE_BOT_TOKEN - Gewebot 访问令牌
  • SMTP_* - 邮件告警配置
